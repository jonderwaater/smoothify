If you want to run stravasmooth (stravasmoothen) locally on your device after installation through "pip install git+https://github.com/jonderwaater/stravasmooth", or after cloning the repository, you have to make your token locally available.

Store your token in a file "~/.stravasmooth" with the following content:

STRAVASMOOTH_TOKEN={{token}}=
